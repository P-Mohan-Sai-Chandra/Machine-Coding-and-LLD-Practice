# Requirements 
***

> **The game should support a configurable board size (N x N) instead of being limited to 3 x 3.**

> **The game should support two players, each assigned a unique symbol (e.g., X and O), and allow alternating turns.**
 
> **The system should validate moves (no overwriting already filled cells and moves must be within board bounds).**

> **The game should detect a winner based on N continuous symbols in a row, column, or diagonal.**

> **The system should detect and declare a draw when all cells are filled and no winner is found.**

> **The game should provide the ability to restart or start a new game without restarting the application.**

> **The Game should support undo operation.**
